## Installation
```sh
$ git clone https://github.com/ZivanovicLuka/GameOfLife-FP
$ cd GameOfLife-FP
```

### Install libs
#### range-v3
```sh
$ cd libs
$ git clone https://github.com/ericniebler/range-v3
```

#### SFML
Using package managers
##### apt
```sh
$ apt install sfml
```
##### brew
```sh
$ brew install sfml
```
##### others
For others look at <a href="https://www.sfml-dev.org/download.php">SFML website</a>

## Running
```sh
$ make
$ ./build/game_of_life
```
